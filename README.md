# כְּזֹהַר הָרָקִיעַ - דף נחיתה

> קומפלקס רוחני-קהילתי בנתיבות · בנשיאות הרב שלום יוסף ברבי שליט״א · עולמות יחד

## 🎯 על הפרויקט

דף נחיתה מודרני, חם ומזמין לגיוס תרומות עבור פרויקט **"כזוהר הרקיע"** — קומפלקס רוחני-קהילתי ייחודי בלב נתיבות, המיועד לציבור המסורתי הרחב, עם דגש מיוחד על **חיילים בודדים** ו**ילדים עם צרכים מיוחדים**.

---

## ✅ פיצ'רים שהושלמו

### 🎨 עיצוב ו-UX
- **עברית מלאה (RTL)** עם טיפוגרפיה איכותית (Heebo + Frank Ruhl Libre)
- **פלטת צבעים מודרנית**: תכלת עמוק (Primary Teal/Azure) + מבטאי נחושת/זהב חם
- **עיצוב responsive** מלא - מותאם למובייל, טאבלט ודסקטופ
- **אנימציות מתוחכמות** עם AOS (Animate On Scroll)
- **מונה מספרים מונפש** לסטטיסטיקות המפתח
- **כפתור צף לתרומה** שמופיע בגלילה

### 📋 תוכן (מבוסס אך ורק על הקבצים המקוריים)
- **Hero Section** — עם שם הפרויקט הנכון "כזוהר הרקיע", פסוק "כי לעולם חסדו", ונשיאות הרב שלום יוסף ברבי שליט״א
- **חזון** — דגש על חיילים בודדים וילדים עם צרכים מיוחדים **בלב הפרויקט**
- **8 הפעילויות** — בדיוק לפי המסמך:
  1. כיתות סיוע לילדים בעלי צרכים מיוחדים *(מודגש)*
  2. חדרי אירוח פרימיום לחיילים בודדים *(מודגש)*
  3. בית כנסת מפואר עם מניינים יומיים
  4. מקוואות מפוארים לנשים ולגברים
  5. מתחם ייעוץ אישי, זוגי ומשפחתי
  6. חלוקת מזון וארוחות חמות לנזקקים
  7. ייעוץ רפואי חינם ע״י ד״ר ופסיכולוג
  8. כוללים ושיעורי תורה בכל הרמות
